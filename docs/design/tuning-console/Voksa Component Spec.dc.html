// state model + engineering contracts
paths: k.<knob> (16) · a.<cmavo>.<i> (7×8) · v.<phoneme>.<i> (377, flat per-item arrays in VoiceTable::to_array order)
ranges map: path → {min,max,step:'any'} exists ONLY while widened; reset/apply-descriptor removes it (setSliderExact port)
diffing: f32 space — fround(v) ≠ fround(def); drives dots, Δ counts, delta export
export: flat text/flags/16 knobs always; attitudinals + phonemes delta-only (stops nest closure/burst objects); + phonetics, notes, sampleRate, voksaVersion
load = REPLACE: rebuild from engine defaults, then apply config keys, widening as needed; preset → custom; filename chip; never merge over dirty state
presets: reset-to-defaults + knob overrides. Deviation from reference: FLAGS ARE PRESERVED (old page cleared them) — keep, it is deliberate
auto-speak: every mutator kicks a 400 ms debounce when enabled + engine ready; new render replaces live AudioWorkletNode
defaults: seed from voksa_default_params() at runtime; voksa-engine-data.json here mirrors it for design only
files: voksa-engine-data.json (descriptors/ranges — regenerate from wasm) · voksa-help-text.json (fill copy, keys stable) · assets/voksa-mark.svg · reference logic: Voksa Workbench.dc.html
